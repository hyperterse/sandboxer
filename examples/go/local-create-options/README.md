# local-create-options

Passes optional create fields such as template image, metadata, and environment
variables. Adjusts ``CreateSandboxRequest`` / ``createSandbox`` to show how you
label and configure the machine before commands run.

## Prerequisites

- Go toolchain and Docker; run from ``examples/go``.

## How to run

```bash
cd examples/go
go run ./local-create-options
```
