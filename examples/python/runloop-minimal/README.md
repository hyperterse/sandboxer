# runloop-minimal

Creates a Runloop sandbox and runs ``echo``. Validates your Runloop API key and
base URL configuration.

## Prerequisites

- Python 3.10 or newer; ``RUNLOOP_API_KEY``; optional ``RUNLOOP_API_BASE``.

## How to run

```bash
export RUNLOOP_API_KEY=your_key_here
PYTHONPATH=sdks/python/src python examples/python/runloop-minimal/main.py
```
