# runloop-minimal

Creates a Runloop sandbox and runs ``echo``. Validates your Runloop API key and
base URL configuration.

## Prerequisites

- Node.js 18 or newer (or Bun); ``RUNLOOP_API_KEY``; optional base URL.

## How to run

```bash
export RUNLOOP_API_KEY=your_key_here
npx tsx examples/typescript/runloop-minimal/index.ts
```
