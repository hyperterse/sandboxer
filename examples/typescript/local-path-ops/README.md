# local-path-ops

Creates a directory, checks that a path exists, and removes a path using
``MakeDir`` / ``make_dir``, ``Exists`` / ``exists``, and ``Remove`` /
``remove``. This mirrors common file tree setup before running commands.

## Prerequisites

- Node.js 18 or newer (or Bun) and Docker.

## How to run

```bash
npx tsx examples/typescript/local-path-ops/index.ts
```
