# list-sandboxes-filter

Calls ``ListSandboxes`` / ``list_sandboxes`` with a ``limit`` and optional
metadata filter so you can narrow results when many sandboxes exist.

## Prerequisites

- Node.js 18 or newer (or Bun) and Docker.

## How to run

```bash
npx tsx examples/typescript/list-sandboxes-filter/index.ts
```
